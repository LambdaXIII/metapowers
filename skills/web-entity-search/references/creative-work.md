# 作品模板

📍 `web-entity-search` · 搜索电影、剧集、书籍、游戏等创作产物 · [← 回 SKILL.md](../../SKILL.md)

## 搜索指引

首选来源：
- 中文：豆瓣、百度百科、时光网（影视）、Bangumi（动画/游戏）
- 英文：IMDb、Rotten Tomatoes、Metacritic、Goodreads（书籍）、Wikipedia
- 一手来源：官方页面、发行方公告

避坑：
- 评分≠地位——IMDb 8.5 和豆瓣 9.0 含义不同，不要跨平台对比
- 票房数据注意口径（全球/本土/含重映）
- 游戏销量数据区分"出货量"和"实际销量"

## 维度

| # | 维度 | 必填 | 关键 | 搜索词提示 |
|---|------|:--:|:----:|-----------|
| 1 | 创作者 | ✓ | ★ | `"{名称}" 导演`（影视）/ `"{名称}" 作者`（书）/ `"{名称}" 开发商`（游戏） |
| 2 | 年份 | ✓ | ★ | `"{名称}" 上映` / `"{名称}" 发行日期` |
| 3 | 类型/题材 | — |   | 通常从搜索结果直接可得 |
| 4 | 市场表现 | ✓ | ★ | `"{名称}" 票房` / `"{名称}" 评分` / `"{名称}" 销量` |
| 5 | 一句话地位 | — |   | 综合搜索结果中反复出现的评价、奖项、影响力描述 |

- "一句话地位"不是独立搜索——它来自前几维的综合判断
- 对连载中/未完结的作品标注状态
