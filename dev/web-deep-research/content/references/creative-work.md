---

研究创意作品时，通用线索追踪流程依然适用——但以下方面可以参考不同的做法。

## 标题策略

创意作品有多个名称，用错了搜索结果天差地别。

- **多语言变体**：至少准备三套搜索词——原标题（原文）、官方英译、中文通用译名。日文作品需同时搜罗马音和原文（如 攻殻機動隊 + Ghost in the Shell + 攻壳机动队）
- **年份标注**：初次发行年份是关键的消歧工具。同名作品、重制版、不同媒介的改编版——带不带年份，搜索结果完全不同
- **按信息维度换关键词**：搜制作背景用"制作""staff""behind the scenes"；搜评价用"review""criticism""评价"；搜影响力用"influence""legacy""影响"。不同的信息目标对应不同的配套词

## 来源分类与引导

不是所有来源在各类信息上都同等有效。以下按"要获得什么信息 → 去哪里找"组织。

### 优先来源（必查）

每次研究创意作品时，以下两个来源应在 Phase 2 最早访问——覆盖了最基本的信息维度，且互为补充。

#### 豆瓣（优先）

中文资料最丰富、可访问性最强、社区活跃。适合获取：

- **基本信息**：中文片名、导演、演员、上映日期（各地区）、片长、类型标签
- **故事梗概**：官方剧情简介及用户编写的详细剧情
- **用户评价**：短评（海量，可快速感知口碑倾向）、长评（有深度的用户分析）
- **受众反应**：讨论区中的话题、争议点、不同市场的接受度差异
- **关联作品**：豆瓣的"喜欢这部电影的人也喜欢"、系列作品列表

注意：豆瓣评分存在文化偏好——国产作品、日韩作品、欧美作品各自的评分基准不同，不可直接横向比较。

#### IMDB（权威）

国际公认的基础数据源，信息结构规范、可机读。适合获取：

- **基本信息**：原名、罗马音/拼音、国际演职员表（含角色名）、上映日期（按国家/地区）、片长、技术规格（颜色、声音、画幅比）
- **票房数据**：通过 Box Office Mojo 链接获取各地区票房
- **专业评分**：Metacritic 聚合分（可直链查看各家媒体原始评分）
- **用户评分**：可按人群细分（年龄、性别）的评分分布
- **花絮与幕后**：Trivia、Goofs、Quotes 等结构化条目
- **关联作品**：系列、前传/续集、同一创作者的其他作品

注意：中国大陆访问可能需要网络环境准备；部分亚洲作品（尤其早期）的 IMDB 条目可能信息稀疏。

### 按信息需求选择

优先来源查完后，根据 Phase 0 拆解出的目标维度，选择以下类别中的来源。

#### 权威来源（基本信息与事实数据）

用于获取可验证的事实——演职员表、名称、日期、票房数字、技术规格。

- **官方页面**：制作公司官网、发行商新闻稿、官方社交媒体——用于确认最终的制作阵容和官方声明（宣传期的信息可能与最终成品不同，见验证要点）
- **行业数据库**：ANN（动漫）、Bangumi（ACG）、VNDB（视觉小说）、Box Office Mojo（票房）、Oricon（日本销量）——用于数据精确性和行业标准格式
- **电影节/奖项页面**：戛纳、威尼斯、奥斯卡等——用于确认入选/获奖信息和放映历史

#### 社交媒体与社区（用户评价与受众反应）

用于了解公众口碑、粉丝讨论焦点、不同群体的接受度。

- **豆瓣讨论区 / 小组**：中文用户的主要讨论场域，话题广度大、时间跨度长
- **微博 / 知乎**：实时讨论和深度问答——适合了解近期热度和争议
- **Reddit**（r/movies、r/anime、r/books 等）：英文用户讨论——适合了解国际受众反应
- **粉丝维基**（Fandom、萌娘百科）：信息量大但权威性参差——每条声明都需交叉验证。粉丝社群中的推测常被重复到看似"公认事实"

注意：社交媒体信息属于 Phase 3 中的"意见/经验"类——不可单独作为事实依据，需与权威来源交叉比对。

#### 专业评论（深度分析与批评视角）

用于获取有方法论支撑的分析——职业评论人的长评、学术文章、行业媒体。

- **职业影评人长评**：区别于用户短评——评论人有署名、有发表平台、有可追溯的评判标准
- **行业媒体**（Variety、The Hollywood Reporter、Sight & Sound、电影手册）：行业内部的制作分析和市场评估
- **学术期刊**：电影研究、媒体研究、文学批评期刊——用于理论基础扎实的分析
- **制作幕后纪录片 / 创作者长篇访谈**：一手资料——创作者本人对创作意图和过程的阐述

注意：创作者的自我阐述不等于作品的实际效果或受众的实际体验。将创作者的说法与评论界的分析并列，而不是互相代替。

## 类别查询与发现

当用户没有指定具体作品——如"最近有什么好看的日本动画电影""2024 年值得关注的独立游戏""有哪些关于 AI 的纪录片"——研究策略与已知作品研究不同。此时的目标不是深度分析某一部作品，而是**在类别中筛选和推荐**。

### Phase 0 的特殊处理

类别查询的指称确认不是确认作品名，而是确认类别边界（见 workflow.md Phase 0 指称确认新增条目）。以下问题值得在 Phase 0 与用户澄清（或至少自己标注假设）：
- 时间范围："最近"是近一个月、近半年、还是近一两年？
- 质量标准："好看"意味着什么？用户偏好什么类型/风格？有没有已知喜好的作品可作为参考系？
- 类型边界：剧场版算不算"电影"？OVA 算不算？Netflix 原创动画电影算不算？

### 来源策略

类别查询的信息来源与已知作品研究不同——不再是豆瓣/IMDB 的单作品页，而是**聚合和榜单来源**：

**建立候选池**（Phase 1-2）：
- 按时间筛选：豆瓣"2025 年动画电影"标签页、MyAnimeList 的季节列表（Winter/Spring/Summer/Fall 2025）、Bangumi 的放送时间排序
- 按评分筛选：在候选池中按评分（≥ 7.5/10 或同等水平）过滤，优先评价人数多的（样本量大的评分更稳定）
- 按类型/风格筛选：根据用户偏好或话题焦点进一步缩小

**补充维度信息**（Phase 2，入选作品逐个浅查）：
- 基本信息：片名（多语言）、导演、制作公司、上映日期、片长
- 一句话梗概：不展开深度分析——够判断是否符合用户需求即可
- 关键标签：类型、风格、目标受众
- 1-2 条代表性评价：选自职业影评或高赞用户短评——用以支撑"为什么推荐"

### 输出节奏

类别查询的产出是**精选列表而非深度报告**。Phase 2 的停止条件比已知作品研究更早到来——当候选池覆盖了时间范围内的主要作品、且非候选作品的信息增量趋近于零时，即可进入 Phase 3 筛选。Phase 4 的报告格式更接近"推荐列表 + 入选理由摘要"，不必套用完整的研究报告模板。

类别查询中的"验证要点"重点不同——不再是验证某一作品的细节是否准确，而是验证：候选池是否遗漏了重要作品？评分标准是否跨文化可比（豆瓣评分 vs MyAnimeList 评分基准不同）？入选理由是否有具体来源支撑而非纯主观判断？

## 研究维度参考

以下维度是可能的切入角度——不是每项研究都必须逐条覆盖。把它们当作"你还可以往这儿想"的提示，而非核对清单。从 Phase 0 确定的目标出发，选择相关的维度深入；其余维度可以跳过，或仅做最浅层的确认。

维度按信息的客观性梯度分三层。层数越高，信息越依赖判断而非事实——对应的搜索策略和证据标准也不同。阅读时留意：你是在找"发生了什么"、还是在问"这意味着什么"、还是在了解"人们觉得怎样"——这三者所需的来源和验证方式完全不同。

### 第一层：客观事实为主

这类信息有相对确定的答案。不同来源之间可能因统计口径或分类体系不同而有出入，但核心是可对齐的事实——不需要判断"哪个说法更好"，只需确认"哪个说法来自哪个来源"。

**基本本体信息**

它是什么？故事梗概、类型标签、风格定位、时长、语言、分级、发行日期、发行地区。

故事梗概各来源的详略和侧重点不同——这不叫分歧，叫编辑选择。类型标签在不同数据库中分类体系可能不一致（豆瓣的"剧情"可能对应 IMDB 的"Drama + Romance"），记录差异即可，不必裁定哪个"正确"。版本差异（导演剪辑版、地区删改版）也归入此处——基本信息的变体。

**商业表现**

花了多少、赚了多少？制作成本、票房、销量、收视率、流媒体播放数据、衍生收入。如果研究对象是出版物或游戏，对应的商业指标不同——但核心问题一致：它作为一个商品，市场结果如何。

数字本身是事实，但不同来源（Box Office Mojo、Oricon、官方财报、媒体报道）的数字可能不一致。标注你采用的是哪个来源、为什么选择它。至于"为什么票房失利""为什么成为爆款"——这类问题已经越界到分析和意见，属于第三层。

### 第二层：事实与评价各半

这类信息中，一部分是可查证的事实，一部分是建立在事实之上的判断。将它们分开——先确认事实骨架，再标注评价来自谁。

**主创人员**

谁导、谁编、谁演、谁制作——这些是事实。核心主创在行业中的大致位置也可以从作品履历中归纳。但"此作品是否其代表作""是否体现了一贯风格""对后续职业生涯有何影响"——这些需要多方评价才能支撑，不是一个人能"查"出来的。

注意：不要陷入完整演职人员名单的陷阱。追问的是核心主创与这部作品的关系——名单是手段，不是目的。通常 3-6 个核心人物就足以勾勒一幅图景。

**创作起源与改编**

是原创还是改编——事实。如果是改编：原作是什么、原作的基本情况——事实。改编策略（忠实还原、结构性重构、仅借用设定）——分析，可以自己归纳但需标注依据。改编效果如何、原作爱好者对改编的评价——意见，应引用具体评价来源，不自己代为判断。

不同版本之间的差异（地区剪辑差异、重制/复刻版的改动、不同媒介的改编取舍）也归入此处。版本差异本身是事实；哪个版本"更好"是意见。

### 第三层：几乎全主观

这类维度本质是感受、判断和阐释。它们可以成为研究报告中最有价值的部分——但也最容易膨胀。如果 Phase 0 的目标没有明确覆盖这些维度，**仅纳入最显著的内容**：被广泛引用的评论共识、公认的名场面、被反复致敬的遗产。不要逐篇挖掘评论、不要追踪争议的完整脉络、不要为了"全面"把报告写成专著。如果某个主观维度恰好是研究目标的核心，那当然值得深挖——但这是有意识的选择，不是惯性牵引。

**艺术特色与亮点**

它好在哪？叙事结构、视听语言、表现手法上的突出之处。哪些场景让人过目不忘？哪些桥段被反复讨论和引用？

区分两个不同的问题："哪些场景被提及最多"是可以客观统计的——这是关于"人们觉得怎样"的事实。"实际艺术水平如何"是阐释——两者是不同的命题。先把前者统计清楚，再归纳后者被提及的理由。

**批评与接受**

评论界怎么说？观众怎么说？不同市场、不同时期的接受度有何差异？

影评共识度（烂番茄新鲜度、Metacritic 均分等）是"关于意见的事实"——它们告诉你评论界的大致倾向，但本身不是一篇评论。每篇评论的具体论点和依据——是意见。不同市场接受度差异的存在——是事实；为什么存在差异——是需要论证的分析。

**影响与遗产**

它留下了什么？长期社会影响、对文艺界或行业的贡献、对后世作品的启发、在类型发展史中的位置。

这几乎纯粹是分析和判断。但辅助手段是存在的："被多少后人引用/致敬/提及"可以客观统计——用统计数据替代空泛断言。"影响了谁、被谁致敬"有据可查；"影响有多大、贡献有多深"永远是阐释，需要论证而非陈述。

## 跨文化/跨市场评价对比

当研究课题涉及同一作品在不同文化/语言市场中的接受度差异时（如"《三体》在国际科幻界的接受度"），以下策略可以参考：

### 分离评价体系

不同市场的评分平台有不同的用户群体和评分文化——豆瓣 8.5 和 IMDB 7.2 不意味着"中国人更喜欢"，可能只是因为两个平台的评分基准和用户构成不同。**对比的不是分数高低，而是评价的内容和模式**。

- 先独立整理每个市场的评价——中文评论界怎么说、英文评论界怎么说、日文评论界怎么说——各自形成完整的画像，再进行横向对比
- 关注各自评价中**聚焦的侧面**不同——A 市场热议的主题可能 B 市场几乎不提及，这个差异本身就揭示了两边对作品的关注点不同
- 注意翻译媒介的影响——非原语言市场的受众读的是译本，译者本身就是第一层阐释者。关注译者在目标市场的地位和翻译策略是否影响了接受度

### 区分不同层面的差异

跨文化评价的差异可能来自不同层面，不要急于归因：

- **信息层面**：不同市场的受众掌握的信息量不同——原产地的观众可能熟悉作品的产业背景和创作语境，海外观众可能只接触作品本身。这不叫"偏见"或"误解"，是信息不对称
- **审美传统层面**：不同文化的叙事传统和审美标准可能使同一手法在不同市场被不同解读。非线性叙事在法国新浪潮的传统中是一种常规，在好莱坞商业片的语境中可能是"大胆创新"——两者都不是"对"或"错"
- **价值观层面**：涉及政治、历史、性别、种族等议题时，不同市场的评价可能反映了根本性的价值分歧。这类差异是重要的研究发现——不需要裁定，但需要标注

### 方法论注意

- **非英语市场的评价更难获取**：英文评价最容易搜索到，但别因此默认"英文世界的评价 = 国际评价"。如果课题要求覆盖非英语市场（如日本、韩国、拉美），在 Phase 1 就规划多语言搜索词，别等到 Phase 3 才发现缺失
- **避免"文化输出"叙事陷阱**：一个作品在海外被喜欢≠"征服了海外市场"——可能只是在特定亚文化圈层中有影响。用具体数据（销量、评分人数、讨论规模）替代空泛的"国际影响力"叙事
- **翻译的不可逆损失**：语言层面的细微之处（文字游戏、韵律、文化典故）可能在翻译中完全丢失。当某个市场对作品的评价集中在"文笔"上时，提醒自己：他们评价的是译本的文笔

## 验证要点

- **官方声明 ≠ 最终事实**——宣传期的制作阵容可能在后续发生变动。宣传材料中的声明需要与最终成品对照
- **票房/销量数据来源不一致**——Box Office Mojo、Oricon、官方财报三者的数字可能各不相同。记录你使用的是哪个来源、为什么选择它
- **访谈翻译的误差**——创作者访谈从日文→英文→中文可能经过两层翻译。尝试找到访谈的原始语言版本，或至少标注你所读的是哪层转译
- **粉丝推测的扩散**——社群中长期流传的"幕后故事"可能起源于某条推文或论坛帖子。追溯信息的最早出处：是官方渠道发布的，还是某个粉丝账号说的？
- **"公认"需要证据**——说"公认的名场面""广泛认为的代表作"时，自查：有多少来源这么说了？是不同的独立判断，还是互相引用？如果只有一两个人说过但被大量转发，那不叫"公认"
