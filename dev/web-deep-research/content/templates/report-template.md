# Research Report Template

📍 `web-deep-research` · 通用研究报告模板 · [← 回 SKILL.md](../SKILL.md)

> **核心原则**：结论前置。读者先看完整的研究成果，需要验证时向后追溯——结论 ← 发现 ← 资料摘要 ← 来源 URL。链条中没有断点。
>
> **章节标题不编号**——编号在写作时是负担（插删一节全部重排），读者也用标题而非编号导航。

## 资料来源与摘要

逐条列出 Phase 2 收集到的所有关键来源。每一条包含：

- **编号**：`[S1]`, `[S2]` ...（全文使用的唯一来源标识）
- **标题/名称**
- **URL**：完整地址（必须可追溯）
- **类型**：事实 / 数据 / 知识框架 / 意见 / 经验

- **摘要**：用自己的话概括来源的核心内容。具体、完整——保留数字、日期、

  不要压缩成"讨论了某话题"式的笼统描述。

## 交叉比对与发现

按 Phase 3 的分类逐类比对。每条发现标注唯一编号 `F1`, `F2` ...，
并注明所依据的来源。

### 事实比对

**F1** X 成立于 2023 年
    [S3] 官方注册信息显示成立日期为 2023-03-15
    [S7] 36氪报道称"2023年初创立"
    [S12] 行业报告中列在"2023年新进入者"分类下
    → 三来源独立一致，高置信度

**F2** Y 产品定价 29 元/月
    [S5] 官方定价页标注 ¥29/月（一手来源）
    [S11] 第三方评测文章称"月费 35 元"（权威二手，可能已过期）
    → 采信 [S5]（一手来源，实时数据），标注 [S11] 存在分歧

**F3** "Z 是市场份额第一"
    [S8] 文中声称 Z 为"行业领导者"，但未提供数据来源
    → 单来源，无独立数据支撑，未赋置信度

### 数据溯源

**F4** 市场规模 504 亿元
    [S1] 来源：中国网络视听协会。方法：平台分账+品牌植入+用户付费三部分
    加总。时间点：2024 年度。不含出海收入。
    [对比] [S9] 引用艾瑞数据为 430 亿——差异因艾瑞口径不含品牌植入。
    → 数据可追溯到权威测量方法，标注口径差异

### 知识 / 框架组合

**F5** Egri 角色骨骼理论与 Save the Cat 节拍表的关系
    [S4] Egri《编剧的艺术》：角色欲望→冲突→转变（角色层面理论）
    [S13] Snyder《Save the Cat》：15 个情绪节拍（结构层面工具）
    → 层级关系：Egri 的角色理论可嵌入节拍表的角色节拍中，互补不冲突

### 意见 / 立场归纳

**F6** 关于 AI 编剧前景的两派意见
    立场 A："AI 是工具而非替代品"——持有者：从业者群体（[S2] [S8] [S15]）
    论据：情感共鸣是人的领域；观众对 AI 内容有信任壁垒
    立场 B："AI 将在 3 年内替代 60% 编剧工作"——持有者：技术乐观派
    （[S4] [S10]）论据：成本降 90%；已在短视频领域验证
    → 争议本质：非事实争议（"AI 能做到什么"可验证），而是价值观争议
    （"什么算创作"不可验证）。标注争议性质，不强行裁定

### 经验可迁移性

**F7** AI 辅助写作降低对白情感浓度的经验
    [S6] 某团队实践：AI 辅助写作后对白情感浓度下降，观众完播率降 15%。
    情境描述：短剧、AI 辅助写作、对白环节、完播率指标（具体可评估）。
    → 可迁移，适用条件限于"对白生成环节"
