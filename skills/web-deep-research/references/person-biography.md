# Public Figure Research

---

研究公众人物时，通用线索追踪流程依然适用——但以下方面可以参考不同的做法。

## 姓名策略

同一人物在不同语境下可能有完全不同的名字。

- **搜集所有写法**：本名、笔名、艺名、英文名、罗马音拼写。尽可能搜全所有形式，优先使用该人物最官方的名称
- **身份限定词过滤**：加"导演""CEO""作者""演员"等限定词，过滤掉同名的无关人物
- **重名验证（贯穿全程）**：公众人物的姓名冲突率远高于一般搜索——即使用了全名+身份限定词，搜索结果仍可能混入同名的其他人。Phase 1 首次搜索后，用额外属性交叉比对并锁定目标：性别、国籍/地区、活跃时期/出生年份、主要职业领域、关键生平节点。这些属性来自 Phase 0 的用户输入或百科条目简介——快速对照即可。若上述属性仍不足以确定目标人物，向用户确认。此后每打开一个新来源，先扫一眼其中的人物基本信息（出生年、国籍、职业简介），确认是本人再继续阅读——不要默认搜到的都是同一个人
- **按时期划分**：人物生涯跨度大时，按时期搜索——"早期""XX 年代""转型前""退休后"等。不同时期的信息分布在不同来源中

Phase 1 结束时必须完成目标人物身份的锁定——通过搜索验证、上下文推理或向用户询问，三种手段任一均可。按上述策略搜集名称变体、添加限定词搜索、用额外属性交叉比对。若穷尽以上手段仍无法锁定，中止研究并向用户确认（给出找到的多个同名人物及其区分特征，让用户选择）。

## 来源分类与引导

不是所有来源在各类信息上都同等有效。以下按"要获得什么信息 → 去哪里找"组织。

### 优先来源（必查）

每次研究公众人物时，先建立基本事实骨架。流程分两步：

#### 第一步：百科条目

**百度百科**——中文世界最便捷的起点：

- 基本信息：中英文名、生卒日期、籍贯、学历、职业标签
- 时间线：按年份排列的履历节点
- 作品/成就列表
- 中文媒体的引用链接

注意：百度百科对在世人物可能受公关维护影响；涉及争议事件时信息可能不完整或存在偏向。

**维基百科（中文/英文）**——信息结构更规范、引用链更完整：

- 基本信息：与百度百科互补，尤其是国际人物
- 引用追溯：维基百科的脚注链接是可追溯的——顺着引用链可以找到原始报道
- 不同叙事视角：对华人公众人物，维基百科可能有不同于国内叙事的描述

两者并查，互相补充，不要单靠一个。维基百科在中国大陆可能无法直接访问。

#### 第二步：领域数据库

查完百科后，根据人物的职业领域，加载对应的数据来源。以下为常见领域：

| 领域 | 数据库 | 提供什么 |
|------|--------|----------|
| 影视/表演 | IMDB、豆瓣 | IMDB 提供国际标准数据（作品年表、角色列表、合作者网络）；豆瓣提供中文作品关联和用户评价 |
| 文学/写作 | Goodreads、豆瓣读书 | 出版年表、作品评分与评论 |
| 音乐 | Discogs、MusicBrainz | 发行年表、合作记录 |
| 学术 | Google Scholar、知网 | 论文列表、被引数据、合作者网络 |
| 商业 | Crunchbase、企查查 | 任职履历、企业关联 |
| 体育 | 赛事官网、Sports-Reference | 成绩记录、转会历史 |
| 政治 | 政府官网、维基文库 | 官方履历、公开声明 |

领域数据库提供的是结构化的客观数据——这是人物研究的"骨架"。

导演、演员、编剧等影视从业者，除 IMDB 外还应查豆瓣人物页——豆瓣的作品关联（同一导演的其他作品、常合作的演员、系列/前传/续集）比 IMDB 更直观，且包含中国大陆市场的发行信息和用户评价。这与创意作品特化文档中将豆瓣列为优先来源的出发点一致：研究作品时必查豆瓣，研究创作者时同样不应跳过。

对于当代公众人物，还应直接访问其社交媒体账号——微博、X (Twitter)、Instagram 等。社交媒体在三个方面不可替代：最新的近况与动态（其他来源更新滞后）、本人或其团队的直接发声（未经二手转述）、以及通过关注/互动/转发关系可能发现的人物关联（配偶、合作者、同行圈子）。注意：社交媒体的信息属于 Phase 3 中的"经验"或"意见"类——本人发布的内容也未必是完整事实（经过形象管理），但作为一手材料具有独特的参考价值。对历史人物或已故人物则无此来源。

### 按信息需求选择

优先来源查完后，根据 Phase 0 拆解出的目标维度，选择以下类别中的来源。

#### 权威来源（基本事实与履历）

用于获取可验证的事实——出生日期、学历、任职履历、获奖记录。

- **官方档案**：政府公开信息、学校校友录、公司年报中的管理层介绍——数据可靠性最高，但覆盖范围有限
- **行业年鉴/名录**：Who's Who 类人物辞典、行业协会会员名录——信息经过编辑审核
- **奖项/荣誉页面**：诺贝尔奖官网、奥斯卡官网、行业奖项数据库——确认获奖/提名信息和日期

注意：官方来源的信息可能经过本人或其代理人审核——准确性高，但信息可能经过选择性呈现。陈述"某人在 X 公司担任 Y 职位"和陈述"X 公司宣称某人担任 Y 职位"是两种不同的精度。

#### 一手与见证资料（自述与他者视角）

用于了解此人的自我定位和同时代人对他的直接观感。这也是验证"权威来源"中事实的关键比对材料。

- **自述资料**：自传、长篇访谈原文、演讲实录、日记/书信（如公开发表）——了解此人如何看待自己、如何叙述自己的生涯
- **同时代见证**：同时代人的回忆录、当时的新闻人物报道、口述历史项目——了解此人在其时代中被如何描述
- **工作产物**：作品本身、公开声明、政策文件——了解此人的实际产出而非他人总结

注意：自述不等于客观事实（见验证要点）。同时代见证受到人际关系、年代视野、信息不对称的影响——一个同时代人的评价不是"历史的定论"，而是"当时的一种声音"。一手与见证资料属于 Phase 3 中的"经验"类——其价值在于提供了事实和权威来源之外的视角，但不可替代权威来源中的事实。

#### 学界与评论界

用于获取有方法论支撑的评估——学者的传记研究、行业史中的定位分析。

- **学术传记**：区别于通俗传记——学术传记有引用规范、有史料考据、有明确的分析框架。通常可在 Google Scholar、知网、JSTOR 上找到
- **行业/领域史**：该人物所处行业或领域的历史著作——将此人的贡献放在宏观脉络中审视
- **批评/评论研究**：针对此人的作品或公共行为的系统批评——不同于新闻报道中的即时评论
- **讣告（如适用）**：大型媒体的讣告通常由专门团队提前准备，包含该人物的核心事实和主流评价——是"一时定评"的快照

注意：专业评估不等于"权威意见"——不同学派、不同时代的学者对同一人物可能有截然相反的评价。评估本身也是需要被评估的。你是在收集研究者的判断，不是在找到"正确答案"。

#### 公众舆论（媒体形象与接受度）

用于了解此人在不同时期、不同群体中被如何认知。

- **新闻档案**：按时期检索大报和通讯社的人名报道——观察报道的基调如何随时间变化
- **中文舆论场**：微博、知乎、豆瓣——了解华语世界对此人的讨论热点和认知变迁
- **国际舆论场**：Reddit、X (Twitter)——了解国际受众的不同视角
- **粉丝/追随者社群**：后援会网站、粉丝维基——信息量大但可靠性参差。粉丝"考证"的细节经常被反复转载到看起来像"公认事实"

注意：公众舆论信息属于 Phase 3 中的"意见"类——反映认知而非事实。舆论的规模（多少人讨论）不等于舆论的正确性。但不同群体之间舆论的差异——这个差异本身是重要的事实发现。

## 研究维度参考

以下维度帮助你建立对"一个人"的清晰认识——不是结构框架，不是核对清单，而是启发提示。调研时自然地按这些方向观察和收集，不必逐条覆盖。

**深度由调研目标决定**——不是每次研究都需要穷尽此人的全部生平。如果调研聚焦于此人与某个事件的关系（如"焦裕禄在兰考治沙中的决策过程"），重心放在生平中与此事件相关的部分、此人在事件中的角色、哪些生平经历影响了事件的走向、以及事件对此人后续轨迹的影响——其余维度的生平细节只做背景了解即可。如果调研聚焦于此人与某部作品的关系，重心是创作过程、作品在生涯中的位置、作品带来的评价变化——不需要为此详列他所有其他作品的票房或评分。五个维度始终是理解一个人的工具，但投入的深度按目标分配，不是平均用力。

### 基本信息与标签

这个人是谁？最直接的辨识信息。

- **名称**：本名、别名、艺名、笔名、花名、俗称——此人被如何称呼。不同语境下可能使用不同名称
- **生卒**：出生日期与地点；如已逝世，逝世日期、地点与死因
- **标签**：用简单的标签描述此人——职业（导演、演员、作者、CEO、通缉犯……可多个）、性别、国籍/地区、活跃时期、特殊身份或气质（如"跨性别者""左撇子""口吃"）。标签不限于"职业"，任何帮助辨识"这是个什么样的人"的特征都可以是标签

基本信息在百科条目中通常已经覆盖。不同来源的生卒日期可能矛盾——标注你采用哪个来源及理由。

### 生平与关键节点

此人的成长经历、职业生涯、人生轨迹。按时间顺序整理，尽量挖掘到最新的近况。

**选取标准**——不是流水账。筛选对人生有影响的事件：

- **关键经历**：出道、成名、转型、争议事件、重大决定、低谷或沉寂期
- **重要成就**：荣誉、奖项、里程碑式的作品或成果
- **重大失误或挫折**：失败的项目、公开的错误、人生低谷——这些往往比成就更能揭示一个人的轨迹
- **转折点**：并不是所有重要事件都是转折点。某个事件是否"改变了此人的轨迹"往往要到回顾时才看得出来——当代的报道和后人的总结可能结论不同。记录事件本身，让评价部分去讨论它是不是转折

**关联人物**——生平中自然会涉及他人：父母、配偶、合作伙伴、导师、对手。在生平中按需提及即可，不展开。详细的关系统一放到「人物关系」一节。

**时代背景**——战争、经济危机、行业变革、制度变迁构成了此人选择的约束条件。不需要单独列一节，在时间线上穿插标注即可。比如"1950 年代（战后重建期）在 X 大学就读"——括号里就是背景。

**证据优先级**：当时的官方公告 > 当时的新闻报道 > 后来的履历总结。后人整理的履历常把时间点写错，追溯原始记录。

### 人物关系

此人与他人的关联。单独列出，但不要过分展开——调研目标是此人，不是他的人脉网。

- **家庭关系**：父母、配偶、子女
- **职业关系**：合作者、竞争对手、雇主/雇员、属于哪个圈子或运动
- **师承/门徒**：师从谁、影响了谁。"XX 的弟子""XX 的接班人"这类标签要追溯来源——是谁说的？自称、媒体加注、还是后人追认？

以研究目标为边界，5-10 个核心关联人物通常足够。不是列表越长越好。

### 外界评价

别人怎么看他？**评价必须与生平关联**——一定是他做了什么事，才会受到这样的评价。不要让评价悬浮在事实之外。

- **不同群体的评价**：粉丝、业界同行、批评家、普通公众——各自可能有完全不同的看法。毁誉参半是常态，不需要裁定"哪边对"，记录分歧本身就是成果
- **重要个人的评价**：权威人士、关键同行、有影响力的评论者的具体评价——不是泛泛的"某人说他好"，而是"谁、在什么场合、说了什么"
- **评价随时间的变化**：此人的声誉是否经历过反转？从何时起、因为什么事，舆论发生了转变？

不同文化/地域对此人的评价可能有巨大差异——一个人在国内被尊为国宝、在国际上几乎无名，或者相反。记录差异，不裁断对错。

注意：评价属于 Phase 3 中的"意见"类。规模（多少人讨论）不等于正确性。但不同群体之间舆论的差异——这个差异本身是重要的事实发现。

### 成果展开

对此人的作品、事件、成就、研究成果等做单独展开——这些往往是调研的核心信息。

- **作品**：导演的电影、作者的书籍、音乐人的专辑、学者的论文——不只列代表作，被忽视的、失败的、中途放弃的同样值得记录
- **事件**：参与的重大事件、策划的项目、涉及的争议——独立展开，提供足够的细节
- **成就与荣誉**：获奖记录、行业里程碑、首创或突破——标注颁奖机构和日期，颁奖机构官网 > 媒体报道 > 本人简历中的获奖列表
- **注意**：发现"最早作品"和"最知名作品"之间的时间差——这段差距本身就是研究材料

成果的类型因人物而异——一个科学家和一个罪犯的"成果"完全不同，但展开的逻辑一致：不只列名字，给出足够的细节让人理解这些成果是什么。

## 验证要点

以下陷阱在人物研究中尤其常见——不是贴士，是在 Phase 3 评估时必须逐一检查的硬问题。

- **自述 ≠ 事实**：自传和访谈中的自我叙述受到记忆偏差、形象管理、提问语境的三重影响。将自述与他述交叉对照——特别关注自述与他述不一致的地方，不一致本身就是发现
- **生平日期经常矛盾**：出生年份、入职月份、作品完成日期——标注你采用哪个来源及理由。不写"此人出生于 1952 年"，写"此人出生于 1952 年（据百度百科，维基百科记为 1953 年，未采用）"
- **争议事件必须多方信源**：仅凭当事人自述、仅凭对立面叙述、仅凭第三方报道——任一种单独使用都不够。三方对不上是常态，记录分歧本身就是研究成果
- **关系标签可追溯**："XX 的弟子""XX 的接班人""XX 的密友"——谁首先使用了这个标签？在什么语境下？是自称、是媒体加注、还是后人追认？标签本身是一个论断，需要查证而非接受
- **"公认"需要独立来源**：说"公认的大师""普遍认为的代表作"时，自查：有多少独立来源这么说了？它们之间是互相引用还是独立判断？互相引用的共识可能是"回声室"，不是"公认"
- **粉丝整理 ≠ 历史记录**：粉丝社群中的"考证帖"经常被反复转载到看似"公认事实"。追溯每一条粉丝整理的"事实"——它的原始出处是可查证的官方渠道，还是某条推文/帖子？
- **当代报道 ≠ 后来的共识**：事件发生时的媒体报道反映了当时的认知——但当时的认知可能是错的（当事人还在世、更多信息尚未披露）。区分"当时是怎么报道的"和"后来怎么评价这件事的"——两者都是有效的 Phase 3 材料，但分属不同的信息类型
