# Classification System Examples

本目录收录个人知识管理领域的主流分类系统，作为设计或调整 journal 分类体系时的参考。

## 使用方式

- `design-classification.md` 的方法论在相应步骤引用这些示例，附一句话摘要
- Agent 在设计中遇到具体问题时，可单独读取相关示例
- 每个示例的第一段是摘要描述，仅需阅读这一段即可判断是否需要全文

## 目录

| 文件 | 系统 | 核心分类维度 | 适合场景 |
|------|------|------------|---------|
| `para.md` | PARA | 按可行动性（Project/Area/Resource/Archive） | 项目驱动型内容 |
| `zettelkasten.md` | Zettelkasten | 按概念（原子笔记 + 链接网络） | 知识积累与发现 |
| `journaling-default.md` | Journaling 默认方案 | 主观/客观 × 长期/短期（4 种子 → 可演化） | AI agent 日记的起点与演化路径 |
| `moc-lyt.md` | MOC / LYT | 按导航优先（索引笔记 + 流式框架） | 大型笔记库导航 |
| `evergreen.md` | Evergreen Notes | 按演化概念（笔记即 API，随时间生长） | 长期知识写作 |
| `gtd-contexts.md` | GTD Contexts | 按可用手段（工具/地点/精力） | 任务执行上下文 |
| `journaling-default.md` | Journaling 默认方案 | 按内容生命周期 + 认知价值 | AI agent 日记 |
