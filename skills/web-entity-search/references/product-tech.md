# 产品/技术模板

📍 `web-entity-search` · 搜索具体产品、软件、硬件、技术方案 · [← 回 SKILL.md](../../SKILL.md)

## 搜索指引

首选来源：
- 产品官方页面、开发者文档
- 科技媒体评测（The Verge、Ars Technica、少数派）
- 行业分析报告（Gartner、IDC，如有）
- 开源项目：GitHub 仓库页

避坑：
- 营销材料的性能声明需外部评测佐证
- 版本号关注时效——搜索时确认是最新版信息
- "智能""新一代"等营销词汇不代表实质差异

## 维度

| # | 维度 | 必填 | 关键 | 搜索词提示 |
|---|------|:--:|:----:|-----------|
| 1 | 类别归属 | ✓ | ★ | `"{名称}" 是什么` `"{名称}" 功能介绍` |
| 2 | 创造者 | ✓ | ★ | `"{名称}" 开发者` `"{名称}" 出品方` |
| 3 | 核心功能 | ✓ | ★ | `"{名称}" 功能介绍` `"{名称}" 怎么用` |
| 4 | 竞品位置 | — |   | `"{名称} vs"` `"{名称}" 同类产品` |

- 技术方案/编程语言/框架：把"核心功能"替换为"核心设计/特点"
- 竞品位置不需要详尽对比——一句话说明它和同类产品的最大区别即可
