# 关键信息摘要示例

## 关键决策

| 决策 | 时间 | 理由 |
|------|------|------|
| 使用 index.md 作为核心索引 | 2024-01 | 通用、简洁、明确 |
| 不规定文件夹分类 | 2024-01 | 保持灵活性，让 Agent 自行决定 |
| 记录格式不强制 | 2024-01 | 避免过度约束，定义原则而非格式 |

## 重要发现

- **查询比记录更频繁**：设计时应优先考虑查询效率
- **首次记录时读取指南**：减少后续 token 消耗
- **deprecated 由 agent 判断**：不强制规定具体标准
