# CHANGELOG

## [1.0.0] - 2026-04-01

### 新增
- **六步流程**：步骤0-5的完整优先级流程
- **自然判断**：步骤0根据上下文灵活判断意图，不依赖关键词
- **终点决定输出**：终点判断决定最终输出形式
- **综合诊断**：步骤4整合静态检查（七维）+ 动态推演
- **方法设计原则**：步骤2不预设解，根据领域设计具体方法
- **自动技巧应用**：元信息清理、一句话定位、边界约束自动应用
- **迭代修复**：步骤5支持最多3次修复循环

### 改进
- **从"提供选项"到"直接决策"**：不再让用户选择，替用户决策
- **从"技巧模板"到"方法设计"**：不套用思维模式/工作流模板
- **从"知识库"到"工艺专家"**：交付结果而非列举知识

### 核心创新
- **不预设解原则**：步骤2根据领域和任务设计具体方法，必须回答"如何做"和"如何做好"
- **场景驱动**：自动识别用户场景，从合适步骤切入
- **诊断与修复分离**：步骤4只做诊断，步骤5决策是否修复
