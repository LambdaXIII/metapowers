# 公司/机构模板

📍 `web-entity-search` · 搜索企业、组织、政府机构 · [← 回 SKILL.md](../../SKILL.md)

## 搜索指引

首选来源：
- 中文：企查查/天眼查（工商信息）、36氪/虎嗅（媒体报道）、公司官网
- 英文：Crunchbase、Wikipedia、TechCrunch、公司官网
- 一手来源优先：官网 About 页、财报/年报、工商注册信息

避坑：
- 客户案例/营销文案中的自述不可作事实来源
- 融资报道注意区分"官宣"和"传闻"
- 行业排名来自单一机构报告时标注来源方

## 维度

| # | 维度 | 必填 | 关键 | 搜索词提示 |
|---|------|:--:|:----:|-----------|
| 1 | 行业归属 | ✓ | ★ | `"{名称}" 行业` `"{名称}" 是做什么的` |
| 2 | 成立/起源 | ✓ | ★ | `"{名称}" 成立于` `"{名称}" 创始人` |
| 3 | 核心业务 | ✓ | ★ | `"{名称}" 产品` `"{名称}" 主营业务` |
| 4 | 规模地位 | — |   | `"{名称}" 规模` `"{名称}" 市场份额` `"{名称}" 员工` |

- 规模地位难以量化时，用"行业地位"描述代替
- 对非营利/政府机构，"核心业务"改为"职能范围"
