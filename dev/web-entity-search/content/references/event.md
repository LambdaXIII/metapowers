# 事件模板

📍 `web-entity-search` · 搜索已发生的事件、运动、灾难、里程碑 · [← 回 SKILL.md](../../SKILL.md)

## 搜索指引

首选来源：
- 维基百科（事件类条目通常有清晰的时间线和多方引用）
- 权威新闻媒体的事件回顾/时间线文章（BBC、Reuters、新华网）
- 官方调查报告（如 NTSB 事故报告、政府白皮书）

避坑：
- 刚发生不久的事件各方报道可能不一致——优先引用多个来源交叉确认
- 事件名称可能有多个变体（如"新冠肺炎""COVID-19""新冠疫情"）——搜索时覆盖主要变体
- 阴谋论/未经证实的说法标注为"存在此类说法但未经证实"

## 维度

| # | 维度 | 必填 | 关键 | 搜索词提示 |
|---|------|:--:|:----:|-----------|
| 1 | 时间地点 | ✓ | ★ | `"{名称}" 时间` `"{名称}" 发生` |
| 2 | 关键参与方 | ✓ | ★ | `"{名称}" 参与者` `"{名称}" 涉及` |
| 3 | 起因·经过·结果 | ✓ | ★ | `"{名称}" 原因` `"{名称}" 经过` `"{名称}" 影响` |
| 4 | 影响/后续 | — |   | `"{名称}" 影响` `"{名称}" 后续` |

- 对于持续中的事件，"结果"改为"当前进展"
- "影响/后续"关注长期影响而非短期反应
