# 通用参考文档

## 格式规范

### 文档结构
所有车间文档应遵循以下结构：
1. 概述
2. 输入
3. 处理流程
4. 输出
5. 参考

### 时间码格式
标准格式：`HH:MM:SS:FF`（时：分：秒：帧）
帧率基准：25 fps

### 文件命名规范
`[项目代号]_[车间缩写]_[版本号].[扩展名]`
示例：`PRJ001_CAST_v01.xlsx`

### 版本标记
- v01-v09：初版迭代
- v10+：稳定版本
- RC1-RC5：候选发布版
- FINAL：最终版

## 术语表

| 术语 | 缩写 | 说明 |
|------|------|------|
| 选角决策表 | CDS | 确定演员角色分配的决策文档 |
| 剪辑决策表 | EDL | 剪辑时间线决策记录 |
| 混音分轨文件 | MTR | 各音轨独立混音文件 |
| 母带文件 | MST | 最终母带输出文件 |
| 交付包 | DEL | 最终交付给客户的完整文件包 |
| 时间码 | TC | 视频时间定位标记 |
| 帧率 | FPS | 每秒显示的帧数 |
| 项目代号 | PID | 唯一项目标识符 |
| 版本号 | VER | 文档或文件版本标识 |

---

## 跨车间引用规则

1. 引用上游输出时，必须注明完整路径：`[上游车间]/[输出文档名称]`
2. 引用术语时首次出现必须给出全称并在括号内注明缩写
3. 引用格式规范时，使用格式：`见通用参考/格式规范/[章节标题]`
