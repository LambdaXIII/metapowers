## 项目概述

**项目名称**：CX Studio Skills

**项目目标**：设计和开发 Agent 技能（skills），用于增强 AI 助手的能力。

**当前状态**：开发中。已完成多个技能的设计和实现。

**主要技术栈**：Markdown、YAML frontmatter、OpenCode 技能规范

**关键约束**：
- 技能主文件必须为 SKILL.md
- name 字段必须与目录名一致
- 所有资源文件从 SKILL.md 出发仅可跳转一层
